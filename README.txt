Instruction for running download_alien_image.py

1. Open Command Prompt on Windows (ensure that python is included in the PATH)
2. In the folder that has the download_alien_image.py, type download_alien_image.py
3. When prompted, enter the url to the subreddit page
4. When "Alien Image Downloaded" is printed on the screen, check the same local folder of download_alien_image.py for a 
   file named "alien_image.png" => That is the alien image
